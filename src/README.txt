----------------------------------------------------------------------------In windows open butterfly.sln in Visual C++ and set additional options in:menu->project->butterfly Project->configuration Properties->C / C++->Command Line option:NULL_MOVE ( enable null move )
  FP_MODE ( enable futility pruning )
     PERFT_MODE ( run perft test ) HASH_MODE ( enable hash table ) HAS_64BITS ( 64 bit architectures ) TEST_MODE ( test mode ) DEBUG_MODE ( debugmode ) In unix use the following command:g++ - w - fno - strict - aliasing - O3 *.cpp - o butterfly - lpthread[parameter] add these parameters to customize it:-DNULL_MOVE ( enable null move ) - DFP_MODE ( enable futility pruning ) - DPERFT_MODE ( run perft test ) - DHASH_MODE ( enable hash table ) - DHAS_64BITS ( 64 bit architectures ) - DTEST_MODE ( test mode ) - DDEBUG_MODE ( debugmode )
     In linux could be compiled with Eclipse  butterfly MODULES:-------------------------|||||cap.cpp ( 1 ) | ||||--------------------------------------------------|||||gen.cpp ( 2 ) | ||||--------------------------------------------------|||||eval.cpp ( 3 ) | ||||--------------------------------------------------|||||search.cpp ( 4 ) | ||||--------------------------------------------------|||||test.cpp ( 5 ) | ||||--------------------------------------------------|||||winboard.cpp ( 6 ) | ||||--------------------------------------------------|||||swapoff.h ( 7 ) | ||||--------------------------------------------------|||||openbook.cpp ( 8 ) | ||||-------------------------Enabled modules whith these compiler parameter:PERFT_MODE:( 1 ), ( 2 ), ( 4 * ), ( 6 )  HASH_MODE:( 1 ), ( 2 ), ( 3 ), ( 4 * ), ( 6 ), ( 7 ), ( 8 )  NULL_MODE:( 1 ), ( 2 ), ( 3 ), ( 4 * ), ( 6 ), ( 7 )  FP_MODE:( 1 ), ( 2 ), ( 3 ), ( 4 * ),
